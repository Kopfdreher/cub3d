## Description
Cub3d is a first person maze based game modeled after Wolfenstein. By completing this project we learned window managment, color rendition, ray-casting, map parsing and more.

## Instructions
The aim of this project is to create a 3D first person representation of a maze using ray-casting. A map is given as a *.cub file, which must be parsed for validity and rendered using ray-casting and math from the math library. The direction that they player is facing is given in this document along with the color of the ceiling and floor and texture of each of the walls. The texture is direction dependent, meaning that West and East for example can have varying textures.
